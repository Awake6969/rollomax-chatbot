# Avatar Image

Place a 64x64px PNG image here named `avatar.png`.
This will be displayed in the chat header.

## Requirements
- Format: PNG (transparent background recommended)
- Size: 64x64 pixels
- File name: avatar.png

## Recommended Design
- RolloMax logo or a stylized sun/blinds icon
- Simple, recognizable at small sizes (32x32px display)

## Fallback
If the image fails to load or is not present, the widget displays "R" in a circle with accent color background.
